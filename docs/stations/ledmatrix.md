# LED Matrix 

Creat fun pixel art with an LED Matrix.  See what kinds of fun art you can build and what cool animations you can create in 10 minutes.

![Project](../assets/stations/ledmatrix/project.png)


<video controls src="https://storage.googleapis.com/noah-education-videos/circuithackingmondays/ledmatrix-station-arduino.mov" ></video>


## Instructions

1. Go to tab [LED Matrix Tab](https://phptuts.github.io/Tools/ledmatrix.html)

2. Create frames you want to display and put in the time you want to have each frame run.

3. Click the copy code button and copy it into [duino.app](https://duino.app/#/code).

4. Upload the code.

## Wiring

| Arduino Pin | LED Matrix Pin | 
| ----------- | -------------- | 
| 5V          | VCC            | 
| GND         | GND            |
| 12          | DIN            |
| 11          | CLK            | 
| 10          | CS / LOAD      |
